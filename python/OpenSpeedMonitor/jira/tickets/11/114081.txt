Create new page "Export/Import" UI (Client)	"*Summary:*

Create the User Interface of the new ""Export/Import""-screen



*Details:*

* no functionality - just UI

* Export on the left half of the screen - Import on the right half



Export:

* Excel and XMI-Format

* Metamodel and Datamodel

* Switch between Full and Partial Export

* If partial switch is active, activate iteraQl-filter (default deactivated)

* ""iteraQL filter"": text input field with description afterwards

* Show always description: ""Example: @costs  > 1000 \n (see documentation)"" / GER:  ""Beispiel: @costs  > 1000 \n (siehe Dokumentation)"" - Link to https://doc.iteraplan.de/pages/viewpage.action?pageId=8685564



Import:

* 3 import strategies (incl. description) as bullets (no dropdown)

* Include metamodel changes as on-off-switch

* Upload-file as dialog like in reactor



Import-Progress:

* Show progress on the bottom of this screen instead of a second screen

* Show info-message about user and strategy

* Show filename and timestamp

* Show all 5 stages and actual status of the stages

* ""Cancel"" and ""Next"" buttons (back to Export/Import main page is not needed anymore)

* Show progress-bar in step 5

* Show information log on bottom
