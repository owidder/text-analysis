Frage 1:

""Visualisieren im interaktiven Client"" bedeutet, das er das Diagram im interaktiven Client mit Standardwerten öffnet. 
 Hier werden *keine* Einstellungen aus der Ansicht im Classic Client *übernommen.*

Um Einstellungen für Diagramm in den interaktiven Client übernehmen zu können, gehen Sie wie folgt vor:
 # Gehen Sie auf ""Visualisierungen"" und wählen das gewünschte Diagramm aus
 # Wenn Sie eine gespeicherte Abfrage an dieser Stelle ausführen können Sie direkt mit dem Punkt 5 fortfahren 
 # Jetzt stellen Sie die gewünschten Filter ein und klicken auf ""Abfrage abschicken und bestätigen""
 # Auf der geöffneten Seite mit den Einstellungen für das Diagramm selbst z.B. Farbe klicken Sie auf ""Abfrage Speichern""
 # Nachdem Sie die Abfrage gespeichert haben, öffnen Sie den Reiter ""Visualisierungen"" und ""Gespeicherte Abfrage migrieren""
 # Auf dieser Seite können Sie gespeicherte Abfragen für Diagramme und Ihre Listenansichten in den interaktiven Client übertragen 
 # Makieren Sie die gewünschten Abfragen und klicken Sie anschließen auf den Knopf ""Migration in den neuen Client"" 
 # Nach Abschluss der Migration wechseln Sie in den interaktiven Client und wechseln in den Tab ""Diagramme und Auswertungen"" und öffnen den Punkt ""Alle gespeicherten Abfragen""
 # Hier finden Sie die migrierten Abfragen die Sie nun im interaktiven Client öffnen können 

Bitte beachten Sie das es bei manchen Abfragen keine 1 zu 1 Übernahme gibt. Unter unserer Dokumentation finden Sie alle Sonderfälle:
