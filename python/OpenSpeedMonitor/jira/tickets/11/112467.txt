Subscribe for one Building Block Instance	"*Summary*

 User is able to subscribe / unsubscribe one Building Block Instance



*Scope1*

 * Dropdown list named ""[Element] Subscriptions"" (german: [Element] Abonnement) with two possible actions (as secondary text-button)

 ** (icon - star) Subscribe /or/ (icon - star border) Unsubscribe

 ** (icon - person) Show subscribers

 * an icon before description in Element Details view (icons should show is-status and description - an action to take)

 ** {color:#14892c}star {color}

 ** {color:#f79232}star border{color}

 * description next to the icon

 ** {color:#14892c}Subscribe to [element name] (german: [Element name] abonieren){color}

 ** {color:#f79232}Unsubscribe from [element name] (german: [Element name] abbestellen){color}

 * after clicking this button user is added to / deleted from subscribers of a list

 * message dialog: (top right corner of a screen)

 ** Subscription successful (german: Abonnement erfolgreich durchgeführt)

 ** Subscription unsuccessful (german: Abonnement fehlgeschlagen)



*UI1*

 * button should be located in top right corner, in the same line as Reset View

 * all buttons as secondary text-button
