Ich habe das Gesamt-Layout der Anwendung etwas umgestellt, sodass das Bundle application.js hinter dem footer landet. In diesem Bundle ist jQuery enthalten. Alle Scripte, die jQuery benötigen müssen danach eingebunden werden. Hierfür ist im Haupt-Layout unter layouts/kickstart_osm.gsp eine pageProperty ""page.include.bottom"" vorgesehen. Jede Page kann jetzt genau einen Block include.bottom definieren, in dem Javascripte geladen werden. Eine Reihe von templates (die tiefer in der Seitenstruktur eingebunden sind) beinhalten ebenfalls jQuery-abhängiges JS. Dieses JS muss jetzt in den Body der enthaltenen Seiten verschoben werden. Um Code-Duplizierungen zu vermeiden muss dieser Code in externe JS-Dateien ausgelagert werden.
