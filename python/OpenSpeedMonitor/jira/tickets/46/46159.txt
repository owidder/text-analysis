Missing Uniqueness Constraint on Custom Dashboards Names	It is possible to create mutliple Custom Dashboards with the same name. This can, in combination with the Copy on Edit hidden feature (see ITERAPLAN-2028), lead to confusion. It might be preferable to have a uniqueness constraint on the names of the Custom Dashboards.
