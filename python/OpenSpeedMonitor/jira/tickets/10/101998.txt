Beim Bearbeiten von Location wird eine falsche Vorauswahl getroffen	Wenn man in der CRUD-View auf "edit" klickt, wird ein Formular angezeigt, wo alle aktuellen Einstellungen vorausgewählt sind. Bei Location kann man einen Wpt server über ein Dropdown-Menü auswählen. Leider ist nicht die letzte Auswahl sondern das erste Element aus der Liste vorausgewählt. Wenn man also auf Edit klickt, nichts verändert und dann speichert, ändert sich evtl. der Wpt server.
