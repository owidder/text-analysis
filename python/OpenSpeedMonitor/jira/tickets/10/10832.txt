Erweiterung der REST-API um Abfrage aggregierter CSI-Werte	"Die bestehenede REST-API soll um vier weitere Methoden erweitert werden:

1)
/rest/\[SYSTEM\]/csi/\[DateFrom\]/\[DateTo\]

Für den übergebenen Zeitraum soll ein Gesamt-CSI-Wert berechnet werden. Die beinhalteten CSI-Rohdaten werden nach Page und Browser gewichtet. 
Es erfolgt keine Gewichtung nach Uhrzeit des Tages.

Rückgabe-Klasse:

{code}
package de.iteratec.ism.rest.response;

/**
 * <p>
 * Der CSI Mittelwert eines Shops/Systems über einen definierten Zeitraum.
 * </p>
 * 
 * <p>
 * ANFRAGE: Name des Systems/Shop UND Zeitraum (wie bei resultsbetween).
 * </p>
 * 
 * @author nkuhn
 * @author mze
 */
public class SystemCSI {

	/**
	 * CSI Mittelwert eines Shops/Systems gewichtet nach Browser und Page.
	 */
	double csiValue;

	/**
	 * Der Zeil CSI Wert.
	 */
	double targetCSI;

	/**
	 * Das Delta der {@link #csiValue} zur {@link #targetCSI}.
	 */
	double delta;

	/**
	 * Anzahl der berücksichtigen Rohdaten.
	 */
	int countOfMeasurings;
}
{code}

2)
/rest/\[SYSTEM\]/csiPerBrowser/\[DateFrom\]/\[DateTo\]

Für den übergebenen Zeitraum soll ein CSI-Wert pro Browser berechnet werden. Die beinhalteten CSI-Rohdaten werden pro Browser nach der gemessenen Page gewichtet. 
Es erfolgt keine Gewichtung nach Uhrzeit des Tages.

{code}
package de.iteratec.ism.rest.response;

/**
 * <p>
 * Die CSI Values von Systemen pro Browser.
 * </p>
 * 
 * <p>
 * ANFRAGE: Name des Systems/Shop UND Zeitraum (wie bei resultsbetween).
 * </p>
 * 
 * @author nkuhn
 * @author mze
 */
public class SystemBrowserCSIValues {

	Collection<SystemBrowserCSI> systemBrowserCSIValues;
	
	/**
	 * <p>
	 * Der CSI Mittelwert eines Systems/Shop für einen bestimmten Browser über
	 * einen definierten Zeitraum.
	 * </p>
	 * 
	 * @author nkuhn
	 * @author mze
	 */
	public static class SystemBrowserCSI {

		/**
		 * Der Name eines Browsers z.B. FF.
		 */
		String browserName;

		/**
		 * CSI Mittelwert des Shops/Systems für einen Browser gewichtet nach
		 * Page.
		 */
		double csiValue;

		/**
		 * Der Zeil CSI Wert.
		 */
		double targetCSI;

		/**
		 * Das Delta der {@link #csiValue} zur {@link #targetCSI}.
		 */
		double delta;

		/**
		 * Anzahl der berücksichtigen Rohdaten.
		 */
		int countOfMeasurings;
	}
}
{code}

3)
/rest/\[SYSTEM\]/csiPerPage/\[DateFrom\]/\[DateTo\]

Für den übergebenen Zeitraum soll ein CSI-Wert pro Page berechnet werden. Die beinhalteten CSI-Rohdaten werden pro Page nach Browser gewichtet. 
Es erfolgt keine Gewichtung nach Uhrzeit des Tages.

4)
/rest/\[SYSTEM\]/csiPerPageAndBrowser/\[DateFrom\]/\[DateTo\]

Für den übergebenen Zeitraum soll ein CSI-Wert pro Browser und Page berechnet werden. Für die beinhalteten CSI-Rohdaten werden ungewichtete Mittelwerte gebildet.

		 * Der Zeil CSI Wert.
