Edit on Single Attribute View - Numeric field (Client)	"*Summary:*

Create Edit-Functionality for Numeric Fields on all Single Attribute Views.



*Scope:*

* In scope: All numeric fields

* In scope: All Single Attribute Views

* Example: ""Lower bound""-field



*UI:*

* Use UI and interactions from Building Block List and Single Building Block View as blueprint (including error-handling)

* If user sets the field to the null value, the field shows ""not assigned"" (German: ""nicht zugeordnet"")

* Ensure that the user enters a number

* Show the number always with two digits (localised)



*Technology:*

* Try to reuse common edit-components from Building Block List and Single Building Block View

* Send REST-request to the correct REST-API (attributes, icresp or icdi)

* Trigger reload-service (IPTS-447)"		New Feature	iteraplan Speedbird	22/Dec/16 9:16 AM	01/Mar/17 5:20 PM
