Parameterauswahl im EventresultDashboard dynamischer gestallten	"Momentan wird einmalig abgefragt, welche JobGroups, Pages, Browser etc zur Verfügung stehen. Der Benutzer kann aus allen wählen. Dabei kann leicht eine Parameterkombination entstehen, für die es keine Daten gibt.



Die Oberfläche soll so angepasst werden, dass nur Parameter angezeigt werden, für die es JobResults gibt. Wenn es im ausgewählten Zeitraum für eine JobGroup keine Daten gibt, soll sie ausgeblendet werden. Es sollen nur Pages angezeigt werden, die in den ausgewählten JobGroups  verwendet werden.  Dies soll für möglichst alle Kombinationen von Zeitraum, JobGroup, Page, Measured Event, Browser, Location und Connectivity erfolgen.



Es muss eine Checkbox geben, die diese Filterung ausschaltet.



Es muss einen Reset-Button geben, der die Auswahl aufhebt und wieder alles anzeigt.
