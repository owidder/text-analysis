Bei einer aktivierten Historisierung werden Änderungen am Datenbestand aufgezeichnet und sind u.a. bei der Anzeige eines Elementes einsehbar. Wenn die Historisierung deaktiviert ist, werden keine Änderungen protokolliert oder angezeigt. Früher aufgezeichnete Änderungen bleiben aber gespeichert.



Historization

Enable data historization



If historization is activated, all changes to the data in iteraplan are recorded. Changes are shown when viewing an element. If historization is disabled, no changes are logged or displayed. Previously recorded changes are nevertheless kept.
