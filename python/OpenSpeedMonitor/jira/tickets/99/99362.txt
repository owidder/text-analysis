RELEASE: REST API for Permission of Admin	"*Summary:*

Create a new REST API to show if you have the permission to ""edit configuration of iteraplan"".



* URI: /api/administration/permissions

* only GET:

* HTTP-status codes:

** 200 = Successfull and JSON with the permissions in the content

** 403 = No permission (put the error message in the body of the response)

** 500 = all other error cases (put the error message in the body of the response)

* JSON-structure:
