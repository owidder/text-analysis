# Bamboo Java Build erweitern



Durch Verwendung von ""failBuildOnCVSS=0"" innerhalb des gradle build, wird ein Abbruch erzeugt, sobald eine Schwachstelle Ausfindig gemacht wird. Durch eine Anhebung des Wertes werden entsprechend größere Schwachstellen (bis max CVSS 10) akzeptiert. Sobald ein Wert > 10 gewählt wird, schlägt der Build-Prozess niemals fehl und die Schwachstellen werden in Logs dokumentiert."
