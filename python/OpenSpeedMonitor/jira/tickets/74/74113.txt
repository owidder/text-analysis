* Wenn man Job-Gruppen ohne weight hinzugenommen hat wird das System beim Klick auf _Create_ nicht übernommen (was korrekt ist). * Die Warnung _Weights have to be of the type double_, wenn man ein weight im falschen Format eingibt sollte statt in einer blauen Box in einer roten (oder gelben) Box stehen.
