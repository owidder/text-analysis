# Jetziges static CSI-Dashboard: wöchentlich shop, Freitag-Freitag, keine angebrochenen Intervalle

# Neues CSI-Dashboard für die Donnerstags-Schnieders-Runde: täglich shop, 4 Wochen Daten



In diesen Dashboards (neue Domain) lassen sich die folgenden Dinge Speichern:

 

* CSI oder RAW

* Zeitraum

* Zeitlichen Aggregation (RAW, HOURLY, DAILY oder WEEKLY)

* Fachliche Auswahl (Jobfilter-Auswahl bei Typ RAW, fachliche Aggregation bei Typ CSI)



Die benutzerspezifischen Dashboards sollten in drei unterschiedlichen Stufen veröffentlicht werden können:

# privat - Diese Dashboards gehören einem angemeldeten User und sind nur für diesen sichtbar.

# privat und für einzelne weitere User veröffentlicht - Diese Dashboards sind für den Besitzer und alle User sichtbar für die sie veröffentlicht wurden.

# öffentlich - Diese Dasboards sind für jedermann sichtbar, auch für nicht eingelogte Gäste.
