Option: "Show orphaned elements"	"*Summary:*

Create a new option to show orphaned elements, i.e. outer elements *without* an inner element.



*Implementation Details:*

* Default is deactivated

* If no inner elements exist, please use for the orphaned outer element the same size as for the inner element.

* Url: Save configuration in URL



*UI:*

* Legend: Show only the activated option in the url:

** New main entry:""Settings"" (German: ""Einstellungen"")

** New sub-entry ""Including outer elements without an inner element"" (German: ""Enthält äußere Elemente ohne ein inneres Element"")

* Toolarea: New subentry for ""Cluster"":
