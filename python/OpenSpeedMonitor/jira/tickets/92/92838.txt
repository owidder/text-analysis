DC-Dashboard keine Dimension aufmachen für eindeutige Werte	Werden Datensätze abgefragt, bei denen ein Wert nach dem gefiltert werden könnte, eindeutig für alle Datensätze ist, beispielsweise wurden alle Messungen mit dem Browser Chrome gemacht, so sollte für diesen Werte keine Crossfilter-Dimension aufgemacht und ein Pie-Chart angezeigt werden. Stattdessen sollte der Fakt, dass für alle Daten gilt, dass sie mit Chrome gemessen wurden, irgendwo auf dem Dashboard angezeigt werden.
