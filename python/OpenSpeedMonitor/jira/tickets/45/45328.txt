WIKI-Syntax for Information System Interface names on the Information System pages	"Under certain circumstances a underscore in the name of an information system interface isn't be displayed properly on the related information system pages. Instead the part of the interface name following the underscore is displayed in italics, affected by WIKI syntax formatting.
WIKI syntax shouldn't be activated for information system interface names."
