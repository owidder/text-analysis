New properties: to activate CC; to start with CC (server)	"*Summary:*

Create 2 new properties in the iteraplan.properties



1. activate.classic.client:

* Default is false

* If true: Create a new link in the user menu (top right) of the New Client

** ""Show in Classic Client"" / GER: ""Zeige im Classic Client""



2. start.with.classic.client:

* Default is false

* *This option has only an effect if activate.classic.client = true* (pre-requisite)

* If true, after login the user gets automatically redirect to the classic client start screen (was already implemented in version 5.5)



{color:red}UPDATE: {color} Please extend the REST-API ""api/administration/generalInformation/"" for this purpose.
