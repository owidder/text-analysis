Good way to set class methods:

{code}

function SomeClass($log, $translate...) {

   this.$translate = $translate;

   ....

}



// We define method outside of constructor, so each time when we create instance of this class (new SomeClass()) we'll just execute the constructor code, but not re-define class method(s)

SomeClass.prototype.someMethod = function(){

}

{code}

 

not so good way to set class methods:

{code}

function SomeClass($log, $translate...) {

   this.$translate = $translate;

   ....

   // defining method in constructor is not a good way, because each time when you call

   //  var someInstance = new SomeClass(...);

   // the code below will be executed and you will define someMethod again and again

   SomeClass.prototype.someMethod = function(){

   }

}

{code}
