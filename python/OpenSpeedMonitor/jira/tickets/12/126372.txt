vielen Dank für Ihre Antwort.

Würden Sie dies bitte einmal in der Datenbank überprüfen. 
 # Öffnen Sie Ihre Datenbank
 # Gehen in die Tabelle „role“
 # Hier sollte der Datensatz „Ihre LDAP Gruppe“ und „iteraplan_Supervisor“ angelegt sein
 # Notierten Sie sich von beiden die ID´s
 # Jetzt gehen Sie in die Tabelle „role_role“ und erzeugen einen neuen Eintrag
 # Die erste ID ist die von ""Ihre LDAP Gruppe““ und danach die ID des „iteraplan_Supervisor“
 # Damit ist die Rolle ""iteraplan_Supervisor"" der Rolle ""Ihre LDAP Gruppe“ untergeordnet

Nachdem können Sie das Menü aufrufen und die Migration durchführen.
