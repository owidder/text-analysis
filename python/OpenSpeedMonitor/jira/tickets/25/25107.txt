Rickshaw: Farben der Kurven vom User selbst konfigurierbar	"Die Farben der im Chart angezeigten Kurven sollen vom User konfigurierbar sein. Dafür gibt es bereits eine Schnittstelle:

RickshawChartCreation.js -> RickshawGraphBuilder.updateColorsOfSeries(args)

In einem Kommentar ist erklärt wie die Funktion zu benutzen ist, wie args auszusehen hat.

Es muss noch ein GUI gebaut werden, welche dann mit Hilfer dieser Funktion die Farben updatet.



*Erste Ausbaustufe:*

Alle angezeigten Kurven werden im Akkordion-Bereich ""adjust chart"" angezeigt mit ihrer default-Farbe.

Diese Farben können mittels color-Picker ausgwählt/verändert werden. Hierfür könnten wir folgende Controls verwenden:

JQuery: https://lauren.github.io/pick-a-color/

Bootstrap: http://www.eyecon.ro/bootstrap-colorpicker/

Ersterer gefällt mir (nach erster Sichtung im Chrome unter Linux) besser.

Es gibt einen Button, mittels dem man alle veränderten Farben auf die Kurven anwenden kann. Hierfür könnte Wilhelms bestehende Funktion verwendet werden (siehe Beschreibung). 



*Zweite Ausbaustufe:*

Im angemeldeten Zustand werden die einer Kurve manuell zugewiesenen Farben gespeichert und bei folgenden Aufrufen im angemeldeten Zustand würden Kurven der selben Bezeichnung initial wieder mit der gewählten Farbe belegt.
