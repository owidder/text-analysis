Add tool area option to show or hide segment labels	"*Summary*
Add option to toolarea to show or hide segment labels

*Details*
* The value of the toolarea entry will be toggled. Two possible options are:
*# show segment labels (in German ""Beschriftungen der Segmente anzeigen"")
*# hide segment labels (in German ""Beschriftungen der Segmente ausblenden"")
* By default segment labels are displayed
* There is no legend entry
* Store in the URL
* Create a ""Display"" top level toolarea entry for this option
